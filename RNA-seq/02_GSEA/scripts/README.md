# GSEA and visualisation scripts

1. run GSEA
  - see [Running a Gene Set Enrichment Analysis](https://docs.gsea-msigdb.org/#GSEA/GSEA_User_Guide/#running-a-gene-set-enrichment-analysis)
  - params:
    - ```Number of permutations``` 1000
    - ```Collapse/Remap to gene symbol``` No_Collapse
    - ```Permutation type``` gene_set
    - ```Enrichment statistics``` weighted
    - ```Metric for ranking genes``` Signal2Noise
    - ```Gene list ordering mode``` descending
2. merge and filter GSEA output
3. visualise GSEA results
