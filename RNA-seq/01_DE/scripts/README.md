# RNA-seq DE

* 00\_Monocultures\_DEA.R - Differential expression analysis R script for the Monocultures experiment
* 00\_Interactions\_DEA.R - Differential expression analysis R script for the Interactions experiment
* 01\_merge\_Monocultures\_Interactions\_DEA\_results.R - R script for merging results of differential expression analyses of the two experiments
* 01\_just-heatmaps.Rmd - R markdown script for heatmaps for specific DE genes
* 01\_just-heatmaps.html - heatmaps for specific DE genes produced by 01\_just-heatmaps.Rmd
